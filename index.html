HTML Table Reference

Common Table Tags

Name Tag Description
Table <table> The parent element for all HTML tables.
Table Head <thead> The set of rows that define the column headers in a table.
Table Body <tbody> The set of rows that contain the table data.
Table Row <tr> The table row container.
Table Data <td> The container for actual table data.
Table Foot <tfoot> The set of rows defining the footer in a table.

Table Attributes

Name Attribute Description
Column Span colspan Defines how many columns a <td> element should span.
Row Span rowspan Defines how many rows a <td> element should span.
